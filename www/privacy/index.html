Privacy Policy

Personal Data

Snap2SheetPro does not collect any personal data from users.

The extension does not gather browsing history, personal information, or any identifiable data.

Third-Party Services

Snap2SheetPro utilizes browser storage mechanisms for storing user settings and data related to job applications. This includes Chrome (Chromium) or WebExtensions Storage Sync API for Chrome users, Web Storage API (localStorage) for other browser users, and User's defaults database for Safari users.

Donations, if any, are processed by third-party payment processors chosen by the organization facilitating donations.

The Extension Website

Snap2SheetPro's website may collect anonymous statistical data, such as page visits and link clicks, for analytical purposes. This information is used to improve user experience and does not include any personally identifiable data.
